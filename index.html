Txtmax Commands

help

List all available commands.

create <filename>

Create a new file, type code in 70 languages, save and quit with :wq.

view <filename>

View the contents of a file with syntax highlighting.

edit <filename> <line>

Edit a specific line in the file.

delete <filename>

Delete a specified file.

run

Run your code quickly without exiting Txtmax with compilers and interpreters like GCC, Clang, Python, Node.js.

files

List all files in the current directory.

search <filename>

Search for files by name.

info <filename>

Get detailed information about the file (name, extension, creation time, modification time, version control).

examples

Show "Hello, World!" examples in multiple languages.

sql

Show example SQL queries for databases like PostgreSQL and NoSQL.

packages

Install packages via package managers (pip, npm, apt, etc.).

api

Test Restful APIs data directly in the editor using cURL.

exit

Exit the Txtmax editor.

Comprehensive Manual

Getting Started

Txtmax is an advanced text editor that supports various file management features and coding tasks, including syntax highlighting, version control, and much more.

File Management

  • Create Files: Use create <filename> to create a new file.
  • View Files: Use view <filename> to view a file with syntax highlighting.
  • Edit Files: Use edit <filename> <line> to edit a specific line.
  • Delete Files: Use delete <filename> to delete a file.

Advanced Features

  • Git version control integration for managing project history.
  • Run code directly without leaving the editor with built-in support for compilers and interpreters.
  • Manage packages easily through built-in package management support (npm, pip, apt, etc.).
  • Test Restful APIs directly within Txtmax using the api command.

Download Txtmax

To get started with Txtmax, follow the installation instructions below:

git clone https://github.com/Calestial2/Txtmax
cd Txtmax
cd versions
gcc txtmax10.c -o txtmax10
./txtmax10
